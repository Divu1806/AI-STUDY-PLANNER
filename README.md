# 📚 AI Adaptive Study Planner (Rule-Based Expert System)

## 📌 Overview
This project is an intelligent rule-based AI system that generates personalized study schedules based on user performance, syllabus completion, and time constraints.

## 🚀 Features
- Personalized study planning
- Priority-based subject scheduling
- Dynamic time allocation
- Performance-based decision making

## 🧠 Technologies Used
- Python
- Rule-Based Systems

## 💡 Outcome
- Improves time management
- Enhances productivity
- Simulates expert-level planning
